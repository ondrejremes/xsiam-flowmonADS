#### Integrations

##### Flowmon ADS

- Initial release of the Flowmon ADS integration.
- Fetch ADS anomaly events as XSIAM incidents.
- **flowmon-ads-perspectives-get** — list configured detection perspectives.
- **flowmon-ads-events-get** — query events by time range and perspective.
- **flowmon-ads-event-get** — retrieve detailed information about a specific event.
